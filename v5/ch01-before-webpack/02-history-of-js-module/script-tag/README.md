# script-tag

## index.js

```js
console.log('Hello, world!');
```

## index.html

```html
<!-- ch01-before-webpack/02-history-of-js-module/script-tag/index.html -->
<!DOCTYPE html>
<html>
  <body>
    <script src="./index.js"></script>
  </body>
</html>
```

使用 `<script>` 標籤的 `src` 屬性引入外部 JavaScript 。
